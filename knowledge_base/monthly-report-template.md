# Monthly Report Template

The monthly report should follow the structure inferred from March 2026 and April 2026 Insurance reports.

## 1. Executive Summary

Include:

- Overall TPV, MPU/FPU/NPU where available.
- MoM performance.
- Main product/category drivers.
- Key incentive or campaign effects.
- Biggest risks and next actions.

## 2. Performance Overview

Recommended table columns:

- Category/Product
- KPI or target, if user provides it
- Actual performance
- Gap
- Achievement rate, if target exists
- MoM
- Comment

If KPI/target is not provided, still report:

- TPV
- Users or transactions
- AOV
- MoM or MTD comparison
- Main driver/comment

## 3. Last Month Priorities

Split into:

- What went well
- What did not go well
- What needs improvement
- Follow-up action

## 4. Incentive and Campaign Review

Use Promotion Summary where possible:

- Final cost
- Acquisition cost
- Retention cost
- TPV
- Cost/TPV
- AOV
- CAC
- CPU
- NPU/FPU
- Promo users
- Promo transactions
- Redemption metrics

Recommendation options:

- Continue
- Optimize
- Stop
- Need more data

## 5. Bottlenecks and Risks

Examples:

- Product decline above 5% MoM.
- Product decline around or above 15% MoM.
- MTD below same-day previous month.
- Traffic source drop.
- Load is healthy but success conversion declines.
- Incentive cost/TPV is too high.

## 6. Roadmap and Recommended Actions

Common themes from historical reports:

- Expand product visibility and package offer.
- Promote subscription or repeat usage.
- Launch or optimize new products.
- Improve tracking for campaign/source attribution.

## Missing Input Handling

On or after the 20th of each month, if KPI or incentive context was not provided:

- Generate data-backed sections from Tableau.
- Clearly mark KPI/incentive target fields as missing.
- Add a short request for the user to provide missing inputs for the final version.
